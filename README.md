# 🌳 레인보우데이즈: 소원나무 이야기

> 소원을 이뤄줄수록 나무가 자란다!  
> 7명의 친구들이 소원나무를 지키는 따뜻한 모험

## 📋 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 버전 | v3.0 |
| 타겟 | 5~10세 어린이 |

## 🌳 세계관

### 핵심 설정
- **소원나무**: 우주 중심의 세계수, 소원을 열매로 맺음
- **소원의 정원**: 소원나무 주변, 7명이 사는 곳
- **소원 열매**: 소원이 담긴 빛나는 열매
- **2명 제한**: 열매 하나에 최대 2명만 진입 가능

### 나무 성장 시스템
| 난이도 | 성장 |
|--------|------|
| ⭐ 쉬움 | 잎사귀 +1 |
| ⭐⭐ 보통 | 가지 +1 |
| ⭐⭐⭐ 어려움 | 큰 가지 + 꽃 |
| 🖤 검은 열매 정화 | 황금 열매! |
